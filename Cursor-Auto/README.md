# SMG Trade Group - Website Rebuild

A professional, enterprise-focused website for SMG Trade Group, built with React, Tailwind CSS, and GSAP animations.

## Design System

### Design Archetype
**Corporate/Enterprise** - Professional, authoritative, and accessible design optimized for B2B inventory liquidation services.

### Typography
- **Primary Font**: Source Sans 3 (Google Fonts)
- **Scale**: Modular type scale with strong contrast (14/16/20/32/56/72px)

### Color Palette
- **Deep Navy**: `#102A43` (navy-900) - Trust and professionalism
- **Near-Black**: `#0A0A0A` - Authority and separation
- **Off-White**: `#FAFAFA` - Content backgrounds
- **Accent Blue**: `#0066CC` - CTAs and highlights
- **Slate Grays**: Various shades for depth and hierarchy

### Layout Strategy
Alternating light/dark sections with generous spacing (96px+ on desktop), clear visual hierarchy, and scroll-triggered animations for engagement.

## Features

- ✅ Fully responsive design (mobile-first)
- ✅ GSAP scroll-triggered animations
- ✅ SEO-optimized content with keyword integration
- ✅ Accessible components (ARIA labels, focus states, semantic HTML)
- ✅ Professional enterprise aesthetic
- ✅ Smooth scroll navigation
- ✅ Contact form with validation

## Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **GSAP** - Animations
- **Lucide React** - Icons
